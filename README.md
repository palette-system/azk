# AZK
<br>
コードを書かずにオリジナルのキーボードを作成できるキーボードファームウェアです。<br>
<br>


| 対応しているボード | 接続タイプ | ファームウェア |
| ---- | ---- | ---- | ---- |
| XIAO nRF52840 | BLEのみ | <a href="https://palette-system.github.io/azk/nrf52840.html">書き込みページ</a> |

<br>
AZKのキーボードをREMAPやVIAのように設定を行うツールです。<br>
<br>
